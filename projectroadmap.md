# Project Roadmap

## Project Name

SafePath

## Project Description

SafePath is a Python navigation app designed to help users compare the safest and fastest travel routes. The goal of the project is to create a simple application where users can enter a starting location and destination, then view route options based on travel time and safety-related information. Instead of only focusing on the fastest route, SafePath is meant to help users make better travel decisions by considering both speed and safety.
